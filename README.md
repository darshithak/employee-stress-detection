# employee-stress-detection
## Overview

This project focuses on detecting human emotions using facial expressions captured through images or video.

## Technologies Used
- Python
- Convolutional Neural Networks (CNN)
- OpenCV

## Working
- Input image is captured using camera
- Face is detected and preprocessed
- CNN model classifies emotion (happy, sad, angry, etc.)

## Features
- Real-time emotion detection
- Image preprocessing
- Emotion classification
